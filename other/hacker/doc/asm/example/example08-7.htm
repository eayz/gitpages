例8.7 在每页的开始处打印“Assember Language”字符串，并空一行才打印其它内容。

解：
.MODEL SMALL
.DATA
　 Title1　DB　0CH, "Assember Language", 0DH, 0AH, 0AH
　 Count　EQU　$ - Title1
.CODE
　　.STARTUP
　　……
　　MOV AH, 1
　　MOV DX, 0
　　INT 17H ;初始化连接在LPT1上的打印机
　　……
　　MOV CX, Count ;待打印的字符个数
　　XOR BX, BX
again:
　　MOV AH, 5H
　　MOV DL, Title1[BX]
　　INT 21H ;调用DOS功能打印字符
　　INC BX
　　LOOP again
　　…… ;可在此打印其它任意字符
　　.EXIT　0
　　END

关　闭
